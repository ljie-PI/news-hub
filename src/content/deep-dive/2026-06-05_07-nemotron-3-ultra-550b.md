## 1. 事件背景

2026 年 6 月 4 日，NVIDIA 在 Computex 主题演讲后正式发布 **Nemotron 3 Ultra**（型号 `NVIDIA-Nemotron-3-Ultra-550B-A55B`），权重同步上线 Hugging Face、OpenRouter、Together AI 与 DeepInfra。该模型在 r/LocalLLaMA 引爆讨论，原帖标题《Nemotron 3 Ultra. 550 billion parameters, 55B active. 1 million context》迅速成为当周热门 New Model 贴。这是 Nemotron 3 系列继 Nano（30B-A3B）、Super（120B-A12B）之后的旗舰版本，也是迄今为止美国厂商发布的最大规模开源权重模型。

## 2. 核心架构

Ultra 采用 **LatentMoE 混合架构**——Mamba-2 状态空间层、MoE 层与稀疏 Attention 层交错堆叠，总参数 550B，每次前向激活 55B（稀疏度 90%）。关键创新点包括：

- **Latent MoE**：每次调用 4 个专家但推理成本仅相当于 1 个专家；
- **Multi-Token Prediction (MTP)**：5 个推测 token 并行预测，推理吞吐显著提升；
- **NVFP4 原生预训练**：大多数线性层在 FP4 精度下训练，仅潜在投影、QKV、嵌入层保持 BF16/MXFP8；
- **1M token 上下文窗口**，RULER 1M 评测得分 94.7。

训练流程为四阶段：预训练 → SFT → RL → 多领域在线策略蒸馏（MOPD）。许可证为 **OpenMDW License Agreement v1.1**，权重、训练数据与配方完全开放，含 GenRM、基模与 NVFP4 量化版四个 checkpoint。

## 3. 社区热议与争议

r/LocalLLaMA 评论区呈两极反应。支持者强调 Artificial Analysis 给出 **48 分** 智能指数，是 gpt-oss-120b（33）、Gemma 4 31B（39）等美国开源模型的明显跨越；DeepInfra 上吞吐超过 **300 token/s**，远高于 DeepSeek、Kimi 同量级 50–100 token/s 的水平。SWE-Bench Verified 71.9、TauBench V3 平均 70.9、GPQA 87.0 的成绩也获认可。

但本地玩家普遍泼冷水：官方最低硬件门槛是 **4×B200 / 8×H100**，普通 128GB Strix Halo / Ryzen AI MAX+ 395 一体机完全跑不动，多人感叹"只能加入观察名单"，转而选择 Nano 30B-A3B。NVFP4 GGUF 暂无法 1:1 转换（缩放因子分离存储），llama.cpp 社区量化进度滞后。性能层面也有争议：Ultra 仍落后中国阵营的 Kimi K2.6（54 分），距离闭源 Claude Opus 4.8（61 分）更远，被讥为"最强美国二线"。

## 4. 行业影响

Ultra 标志 NVIDIA 从硬件厂商正式跨入"模型公司"行列：完整开源权重 + 数据 + 配方 + 训练框架（NeMo），意在用开放生态绑定自家 Blackwell GPU 与 NVFP4 推理栈。MoE + Mamba 混合 + MTP + FP4 的组合验证了"高稀疏度 + 状态空间 + 低精度"路线在万亿级模型上的可行性，为 Llama 5、DeepSeek V4 后续设计提供参考。对国内开源生态而言，这也加速了中美开源模型在 agent 编排、长上下文 RAG 场景的正面竞争。
