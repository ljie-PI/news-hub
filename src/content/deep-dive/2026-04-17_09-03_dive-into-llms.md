# dive-into-llms 深度报告

> ⭐ 30,689 stars (+1,385 today) | 来源：上海交通大学 | 维护者：袁童鑫(Lordog) 等 | 状态：持续更新中

---

## 定位与痛点剖析

《动手学大模型》是一个面向**大模型入门的编程实践教程**，起源于上海交通大学研究生课程《自然语言处理前沿技术》（NIS8021）和《人工智能安全技术》（NIS3353）的讲义。

核心痛点：
- 大模型相关资料要么太理论（论文）、要么太偏工程（框架文档），缺乏"会做研究的人写的可执行教程"
- 中文社区中系统性、高质量的大模型 hands-on 教程极度稀缺
- 学生和研究者面临"懂原理但不知道怎么上手跑代码"的鸿沟

项目**完全免费**，明确定位为公益性质，不绑定任何商业产品。这与国内大量"付费社群+割韭菜"式 AI 教程形成鲜明对比。

---

## 核心架构与技术细节

**11 个主题章节**（含 PPT、README 教程、Jupyter Notebook 脚本）：

| 章节 | 内容 | 技术方向 |
|------|------|----------|
| 微调与部署 | 预训练模型 LoRA 微调、Demo 部署 | Fine-tuning |
| 提示学习与思维链 | API 调用、CoT 推理 | Prompting |
| 知识编辑 | ROME/MEMIT 等编辑方法 | Knowledge Editing |
| 数学推理 | 蒸馏迷你 R1 | Reasoning |
| 模型水印 | 文本水印嵌入/检测 | AI Safety |
| 越狱攻击 | Jailbreak 方法研究 | Red Teaming |
| 大模型隐写 | LLM Steganography | Security |
| 多模态模型 | MLLM 理解与生成 | Multimodal |
| GUI 智能体 | 自动点外卖、购物等任务 | Agent |
| 智能体安全 | Agent 风险威胁分析 | AI Safety |
| RLHF 安全对齐 | PPO + RLHF 实验 | Alignment |

**2025 年新增内容：**联合华为昇腾推出《大模型开发全流程》系列（初级/中级/高级），基于国产硬件平台，覆盖 PPT + 实验手册 + 视频，托管在华为昇腾社区。

**技术栈：** Python, Transformers, PyTorch, Jupyter Notebook，对国内常用 LLM API（通义、DeepSeek、Kimi 等）友好。

本项目的**差异化优势**：
1. 来自顶尖高校研究团队（SJTU BCMI 实验室），内容有学术深度
2. 覆盖 AI 安全、越狱攻击、水印、隐写等**其他教程几乎不涉及**的方向
3. 与华为昇腾的合作使其获得国产化生态背书

## 开发者反馈与局限性

**正面评价：**
- 极高的内容密度：每章均含课件 PDF + 文字教程 + 可运行 Notebook
- "越狱攻击"章节的自嘲式描述（"先从学会攻击开始"）被广泛截图传播
- 研究生课程背景保证了内容的严谨性，不是简单的 API 包装

**局限性：**
- 仍处于 v0.1.0 建设阶段，部分章节内容深度不均
- 昇腾合作方向引发部分开发者对"国产化"路线的关注，GPU 资源依赖可能限制受众
- 没有配套的 Discord/社区论坛，问题反馈只能走 GitHub Issue
- 内容更新频率依赖研究生/教师的空闲时间，可持续性存在不确定性
- 不涉及实际生产部署（如 vLLM、TGI 推理服务），对工程化需求覆盖有限
