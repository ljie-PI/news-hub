# codeforreal1/compressO — 跨平台视频/图片压缩桌面应用

## 1. 定位与痛点剖析

CompressO（发音同"Espresso"）是一个**免费开源的跨平台视频/图片压缩桌面应用**，支持 Windows、macOS、Linux。

解决的痛点：
- **在线压缩工具隐私风险**：需要将文件上传到第三方服务器，数据泄露风险高
- **FFmpeg 命令行门槛高**：普通用户无法直接使用 FFmpeg、pngquant 等工具
- **专业软件昂贵**：Adobe Media Encoder、Handbrake 等工具要么收费要么功能过于复杂
- **批量压缩需求**：需要同时处理大量媒体文件时缺乏便捷的图形界面工具

CompressO 的定位是：**零上传、本地运行、开箱即用**的压缩工具，将专业命令行工具的能力包装成现代化 GUI。

## 2. 核心架构与技术细节

**技术栈**：
- **Tauri（Rust）**：桌面应用框架，提供轻量跨平台壳（不依赖 Electron 的 Chromium，体积更小）
- **React + Vite**：前端 UI
- **压缩引擎（第三方 CLI 工具，平台专用独立二进制）**：
  - `FFmpeg`：视频压缩/转码/裁剪
  - `pngquant`：PNG 有损压缩
  - `jpegoptim`：JPEG 优化
  - `gifski`：高质量 GIF 生成
- **完全离线**：除内置 App 更新检查外，**零网络请求**

**功能亮点**：
- 视频压缩、裁剪（Trim/Split）
- 批量压缩（Batch Compression）
- 视频/音频编解码配置
- 字幕嵌入（Subtitle Embed）
- 元数据编辑（Metadata Update）
- 支持多种格式输出

**安装方式**：
- Linux：`.deb`（Debian/Ubuntu）、`.AppImage`（通用）
- macOS：`.dmg`（Apple Silicon / Intel），支持 Homebrew（`brew install --cask compresso`）
- Windows：`.msi`（64位）

**已知问题**：macOS 未通过 Apple 公证（需 $100/年 Apple 开发者费用，作者拒绝支付），Homebrew 安装版会自动清除隔离属性，直接下载安装需手动运行 `xattr -cr /Applications/CompressO.app`。

## 4. 开发者反馈与局限性

**优势**：
- 使用 Tauri 而非 Electron，包体小、内存占用低
- 完全离线，隐私友好
- Homebrew 支持，macOS 用户安装体验好

**局限性**：
- macOS 未公证，普通用户安装有障碍（需执行终端命令或信任开发者）
- **Debian 13 / Ubuntu 24 暂不支持**（Tauri 依赖库兼容性问题，等待 Tauri 上游修复）
- 功能集相对基础，不支持高级视频编辑（色彩校正、滤镜等）
- 压缩质量取决于底层 FFmpeg/pngquant 参数配置，进阶用户可能希望更多参数暴露
- AGPL-3.0 许可证限制商业集成
