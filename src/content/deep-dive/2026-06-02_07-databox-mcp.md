## 1. 背景

Anthropic 于 2024 年底推出的 Model Context Protocol（MCP）在 2025-2026 年迅速成为 LLM 工具接入的事实标准。各类 SaaS 纷纷推出 MCP server，把自家数据与能力暴露给 Claude、ChatGPT、Cursor 等客户端。**Databox MCP** 在 Product Hunt 当日跻身前列，定位是“在 Claude、ChatGPT 中和你的业务数据对话”。

## 2. 核心机制 / 项目定位

据官方描述，Databox MCP 连接业务数据到 Claude、ChatGPT、Cursor、n8n 等 MCP 客户端。用户可以用自然语言询问营收、营销活动、销售管道等指标，回答以真实数据为依据。背后是 Databox 现有的多源数据集成（Stripe、HubSpot、Google Analytics、Shopify 等 100+ 数据源），MCP 层把其指标 API 重新包装为 LLM 可调用的工具。

## 3. 社区 / 竞品观察

类似形态的产品大量涌现：Linear MCP、Notion MCP、Stripe MCP、Supabase MCP 等。Databox 的优势在于早已聚合多个 SaaS 指标，单一 MCP server 即可覆盖整套 KPI dashboard，省去用户为每个数据源单独装一个 MCP 的麻烦。竞品角度，Cube、Steep、Glean 等也在做语义层 + LLM 接入。

## 4. 行业影响

“BI as MCP”可能动摇传统 BI 工具（Tableau、Looker）的入口地位：当 CEO 直接在 Claude 里问“本月 MRR 同比”，dashboard 的存在感被削弱。MCP 推动数据消费从“看图表”转向“问问题”，对 BI 厂商而言，谁先提供高质量 MCP server，谁就能锁定 LLM 时代的查询入口。
