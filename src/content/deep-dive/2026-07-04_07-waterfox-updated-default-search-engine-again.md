# Waterfox updated default search engine again

## 事件背景
Waterfox 是由 Alex Kontos 开发的隐私向 Firefox 分支，主打去除遥测、保留旧版扩展兼容性。此帖在 r/browsers 引发讨论，主题是 Waterfox 又一次更换了捆绑的默认搜索引擎。这已是该项目多次调整默认搜索的再一回：2019 年它曾被广告科技公司 System1 收购，2023 年又宣布独立，围绕默认搜索的商业合作始终是敏感话题。

## 核心观点 / 产品机制
据社区讨论，这次变动延续了 Waterfox 反复调整默认搜索的模式。对小型分支而言，搜索引擎默认位往往是核心收入来源，主流浏览器多通过与搜索商的分成或竞价获利。推测本次更换或与其自研的 Waterfox Private Search 元搜索方案有关：官方近年试图在隐私承诺与可持续营收之间寻找平衡，把默认搜索从第三方逐步转向自有产品。

## 社区热议与争议点
据社区讨论，观点大致分为两派。批评方认为，反复且悄无声息地更换默认搜索是对信任的侵蚀，尤其考虑到 Waterfox 曾隶属广告科技公司 System1，默认位背后的变现动机让"隐私向"招牌显得可疑；有用户直言发行说明像是把商业变现包装成温情叙事。支持方则较为务实，认为默认搜索分成是小型分支维持开发的必要收入，只要不上传浏览数据、允许用户自行改回，就属可接受的折中。也有中间意见推测，问题不在于是否变现，而在于变更缺乏充分告知与透明度。

## 行业影响与未来展望
据推测，此事折射出独立浏览器普遍的两难：既要坚守隐私定位吸引用户，又需靠搜索合作维持营收。若 Waterfox 能把默认搜索导向自研且透明的方案，或可缓解信任危机；反之，频繁而不透明的变更会持续消耗小众用户的耐心，把他们推向更纯粹的替代品。
