# showmd（141票）

**Product Hunt 排名：#6 | 2026-04-14**

---

## 事件背景

macOS 的 Quick Look 功能（按空格键预览文件）本身不支持 Markdown 渲染——按空格键预览 .md 文件，你只会看到充满 `##`、`**`、`-` 的原始文本，完全无法读取结构化内容。这对开发者、技术写作者、以及大量使用 AI Agent 配置文件（AGENTS.md、CLAUDE.md、.cursorrules）的用户来说，是每天都会遇到的摩擦。

**showmd** 是一款 macOS Quick Look 扩展，主打"最可读的 .md 文件预览方式"，开源且免费。

---

## 核心观点 / 产品机制

showmd 的核心功能是将 macOS Quick Look 变成一个完整的 Markdown 渲染器，而且专门针对 AI 时代的 Markdown 使用场景做了优化：

- **标准 Markdown 渲染**：标题、列表、代码块、表格、任务列表全部正常渲染
- **YAML Frontmatter 解析**：将 frontmatter 解析为可折叠的元数据表格（默认折叠，一键展开），其他预览工具均未实现此功能
- **AI Agent XML 标签渲染**：`<example>`、`<instructions>`、`<thinking>` 等 17+ 个常见 AI prompt 标签，渲染为带边框的标注块而非原始角括号——这是专门为 CLAUDE.md、AGENTS.md 等文件设计的
- **Mermaid 流程图 & LaTeX 数学公式**：原生渲染图表和公式，其他 Quick Look 扩展均显示为代码
- **源码切换**：一键在渲染视图和原始文本之间切换
- **极简安装**：拖入 Applications，扩展自动注册，System Settings 中一个开关搞定

## 社区热议与争议点

1. **"AI 写 Markdown，你该渲染它"**：官网和 PH 帖子的 slogan 引发强烈共鸣，"AI 生成的 CLAUDE.md 看起来一团糟，这个工具正好解决了我最头疼的问题。"

2. **与 QLMarkdown 等老工具的对比**：有用户指出 `sbarex/QLMarkdown` 等老牌开源方案已存在多年，showmd 的差异化在于 AI Agent 标签支持和 frontmatter 处理，被认为是有效的差异化而非重复造轮子。

3. **开源可持续性**：产品免费开源，有用户关心维护长期性，"希望加入 Homebrew 方便安装，现在要手动拖 app。"

4. **Mermaid 渲染性能**：部分用户反映复杂 Mermaid 图在 Quick Look 中渲染略慢，期待优化。

showmd 是一个精准解决特定痛点的小工具，但其背后折射出一个重要趋势：**AI 原生开发场景催生了新的工具需求**。AGENTS.md、CLAUDE.md 这类文件正在成为软件项目的标配，围绕这类文件的编辑、预览、版本管理工具将形成一个新兴的生态。
