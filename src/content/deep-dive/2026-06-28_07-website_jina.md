## How CoWork Works

1.   ### Import Existing Tests

Bring test cases from Jira, TestRail, spreadsheets, or other test management tools. 
2.   
### Build Execution Context

CoWork converts natural-language test cases into structured BDD context. It understands:

    *   Where the test starts
    *   What actions must happen
    *   What outcome should be validated

3.   ### Execute on Real Devices

CoWork builds an execution plan and begins running the test. No scripts required. 
4.   ### Replan When Reality Changes

Unexpected popup? Different screen? Changed flow? CoWork proposes the next best action and requests approval before proceeding. 
5.   ### Complete the Journey

When user input is required, CoWork pauses and asks. When the input arrives, execution resumes. The test continues without losing context.
