# Nothing Ever Happens：永远买 No 的 Polymarket 预测市场机器人

**来源**: GitHub | 319分 142评论 | 2026-04-14

---

## 事件背景

Polymarket 是目前全球规模最大的去中心化预测市场平台，用户可以对各类问题（政治、科技、社会事件）的结果下注。该平台以"市场是最好的预测工具"为理论基础。

然而，Sterling Crispin 开源的这个 Python 机器人 `nothing-ever-happens`，提出了一个极具调侃意味的简单策略：**在所有非体育类 Yes/No 市场上，无脑买入 NO**，理由是"什么大事都不会真正发生"（Nothing Ever Happens）。该项目在 HN 上引发了关于预测市场定价效率、人类认知偏差和另类投资策略的大量讨论。

## 核心观点/产品机制

**机器人工作逻辑**：
1. 扫描 Polymarket 上的独立（非体育类）Yes/No 市场
2. 寻找 NO 价格低于配置上限的市场
3. 批量买入 NO 头寸
4. 跟踪持仓、暴露实时 Dashboard，持久化恢复状态

**技术实现**：
- 纯 Python 异步架构，代码结构清晰（`bot/`, `scripts/`, `tests/`）
- 需要配置 Polygon 区块链节点、私钥、数据库等才能真实交易
- 默认进入模拟（Paper Trading）模式，需三个环境变量同时激活才实盘

**策略假设**：预测市场上，人们倾向于高估"大事发生"的概率（因为 YES 更刺激、更有话题性），导致 YES 定价偏贵、NO 定价偏低，长期买 NO 存在套利空间。

## 社区热议与争议点

**质疑策略可行性（tekno45）**：用户 `tekno45` 直接搬出作者本人的 Twitter 声明——**承认没有正回报**，认为这只是一个"有趣的项目和 meme"，并非真正的盈利策略。

**支持策略逻辑（slg）**：用户 `slg` 援引体育博彩领域的同类发现：在体育投注中，"买 under（偏空）"长期而言往往更优——因为 over（偏多）的结果更刺激，吸引更多投注者，导致其定价系统性偏高。他认为同样的认知偏差存在于预测市场。但他也坦承：这也可能只是过去市场定价不够智能时的遗留现象，随着市场更高效而消失。

**争议核心**：Polymarket 现在是否已经足够高效，以至于这种简单策略无法持续盈利？还是认知偏差（人类对戏剧性事件的过度期待）是永恒存在的市场低效来源？

**法律风险**：另有用户提醒，在部分国家地区，通过程序化方式参与预测市场的合法性存疑，尤其是涉及加密资产的情况。

## 行业影响与未来展望

1. **预测市场有效性的学术价值**：此类"简单策略"开源项目，为研究 Polymarket 定价效率提供了实证工具，学术界正越来越关注链上预测市场数据。
2. **量化交易下移**：过去只有机构能做的市场中性策略，现在任何懂 Python 的人都可以在去中心化预测市场上尝试，这是 DeFi 的民主化体现。
3. **Polymarket 生态成熟度**：越来越多的 bot 进入意味着市场定价更高效，也意味着简单策略的超额收益会持续收窄——这是市场自我纠偏的健康表现。
4. **风险提示**：真实运行需要 Polygon 钱包、私钥管理等，普通用户资金安全风险不容忽视。
