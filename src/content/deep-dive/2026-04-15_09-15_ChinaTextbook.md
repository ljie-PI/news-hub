# 15 · TapXWorld/ChinaTextbook

> ⭐ 2,561 stars / week | 69,337 total | 2026-04-15

## a. 定位与痛点剖析

**ChinaTextbook** 是一个将中国全套小初高及大学 PDF 教材整合并开源分发的 GitHub 仓库，核心立场是**教育资源的民主化**。

项目发起者明确指出了三重痛点：

1. **国内信息获取不均衡**：虽然官方教育网站提供免费资源，但普通人获取渠道仍受限（需要知道具体网址、注册账号等门槛）
2. **劣质商品化**：有人将这些公共教育资源加上私人水印后在平台上销售，本质是对公共资源的寄生牟利
3. **海外华人需求**：身在海外的华人家长希望孩子能接触到国内的教育体系，但正规渠道不完善

额外深意：项目发起者表达了"希望未来出现更多不是为了考学而读书的人"的教育理念，带有一定的教育改革视角。

## b. 核心架构与技术细节

这是一个**内容分发仓库**，不是软件项目，但其技术解决方案值得关注：

**核心技术问题：GitHub 文件大小限制**
- GitHub 单文件上传上限 100MB（超出则拒绝），建议上限 50MB（超出有警告）
- 教材 PDF 文件普遍超过 50MB（部分高中/大学教材达数百MB）

**解决方案：文件分片 + 合并工具**
- 超过 50MB 的文件拆分为每份 35MB 的多个分片
- 命名约定：`义务教育教科书 · 数学一年级上册.pdf.1` + `.pdf.2` + ...
- 配套工具：`mergePDFs-windows-amd64.exe`（双击自动合并）
- 合并工具开源：https://github.com/TapXWorld/ChinaTextbook-tools

**资源覆盖范围**：
- 小学：数学1-6年级（人教版，上下册）
- 初中：数学初1-初3（人教版，上下册）
- 高中：数学（人教A版）
- 大学：高等数学（同济第七版）、线性代数、离散数学、概率论

**下载方式**：
- 国内用户：使用 tchMaterial-parser 直接从官方渠道下载
- 海外用户：直接从本仓库 `git clone` 或按需下载

## d. 开发者反馈与局限性

**正面**：
- 69.3k total stars，是国内 GitHub 用户关注度最高的纯内容仓库之一
- 受众精准：海外华人家长、国内教育工作者、教育资源研究者
- 配套工具（合并程序）设计周到，降低了非技术用户的使用门槛
- Telegram 社区运营活跃

**局限性**：
- **版权灰色地带**：人教版教材受著作权保护，将其在 GitHub 上分发的合法性存在争议，有被 DMCA 下架的风险
- **资源覆盖不完整**：目前主要覆盖数学，语文/英语/物理/化学/生物等科目覆盖有限
- **更新滞后**：教材每隔数年会修订，仓库内容未必是最新版本
- 文件分片设计增加了使用复杂度（需要下载合并程序）
- 仅 PDF 格式，无法直接接入 AI 辅助学习工具（需配合 markitdown 等工具转换）
