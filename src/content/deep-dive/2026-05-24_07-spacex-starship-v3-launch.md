# 深度调研：SpaceX 发射 Starship v3 火箭

## 事件概述

SpaceX 完成了 Starship v3（第三代星舰）的首次试飞。Starship v3 相较于 v1/v2，进行了大幅结构升级与改型：拉长的箭体提供更多推进剂、新一代 Raptor 3 发动机、改良的隔热瓦与栅格翼、以及更接近 "运营版" 的 Super Heavy 助推器。此次首飞被视为 Starship 从 "原型验证" 进入 "准业务运行" 阶段的标志，是 NASA Artemis 月球计划与 Starlink V3 卫星批量部署的关键基础设施节点。

## 技术细节

公开资料显示，v3 主要变化包括：

- **箭体加长**：总高度突破 130 米，推进剂容量较 v2 增加约 25%–30%；
- **Raptor 3 发动机**：进一步去除外部管路与防热罩，提升推力（接近单台 280 吨级海平面推力），并大幅简化生产；
- **结构与隔热**：新一代不锈钢成分与更耐高温的陶瓷瓦，旨在解决 v2 多次再入测试中前缘瓦片烧蚀的问题；
- **回收路径**：助推器继续采用 "筷子塔（Mechazilla）" 接捕；星舰上面级的目标是实现整箭可重复使用与轨道再加注；
- **载荷能力**：完全可复用条件下设计目标 100+ 吨入轨，对扩展 Starlink V3 大卫星与未来 HLS（Human Landing System）月球着陆器至关重要。

## 行业意义

1. **超重型可重复使用进入实用化**：如果 v3 能在数次飞行内稳定回收上面级，将彻底重写运载市场经济学，每公斤入轨成本可望进入百美元级。
2. **Artemis III 月球登陆**：NASA HLS 合同强依赖于 Starship 在轨加注、月面着陆、月面起飞的能力，v3 是这一链条的核心硬件。
3. **Starlink V3**：单星质量与天线尺寸均显著增加，只有 Starship 级运力可在合理成本下完成大规模组网。
4. **国际反应**：中国（长征 9 号、朱雀 3 号）、欧洲、印度等都在加紧推进可重复使用大火箭项目；v3 的成功将进一步拉大领先窗口。
