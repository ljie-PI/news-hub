# DIY 自制软饮料：从零开始做可乐

## 事件背景

程序员兼创客 blinry 撰写了一篇详细的博客文章，记录了从 2020 年开始自制软饮料的完整过程，包括无糖无咖啡因可乐、橙味苏打水和杏仁苏打水。文章带有"实验日志"风格，配有照片，并在 GitHub 上开源了配方（受到 Open Cola 和 Cube Cola 等开源可乐项目启发）。

这类"自己动手做工业食品"的内容在 HN 和 Maker 社区中具有持久的吸引力，反映了技术人群对理解和重现"黑盒"食品工艺的热情。

## 核心观点/产品机制

**可乐配方的核心技术：**

1. **精油调配**（关键步骤）：
   - 橙油 0.75ml + 青柠油 0.7ml + 柠檬油 0.2ml + 肉豆蔻油 0.15ml + 肉桂油 0.08ml + 芫荽油 0.05ml + 薰衣草油 0.05ml
   - 约 2ml 精油可为 9L 可乐提供香味
   - 精油极度浓缩，需戴手套操作

2. **乳化处理**：
   - 加入 2g 阿拉伯树胶（天然乳化剂）+ 4ml 水，用手持搅拌器混合约 5 分钟，形成均匀的乳状混合物
   - 这一步解决了"油不溶于水"的根本问题

3. **焦糖色 + 酸味**：加入 40ml 焦糖色素 + 5g 柠檬酸

4. **甜味剂方案**：作者为避免咖啡因头痛，使用了无咖啡因版本；使用德国常见的环己基氨基磺酸钠+糖精组合代替蔗糖

5. **最终稀释比例**：1:8 兑入苏打水

**作者最终感受**："它很好喝！非常好！甜度刚刚好，香味组合也很可乐！"

## 社区热议与争议点

HN 239 条评论涵盖了食品科学、化学知识和怀旧情绪：

1. **Open Cola 和 Cube Cola 的历史讨论**：许多用户提到这两个开源可乐项目，讨论开源配方运动的历史。有人分析可口可乐的"神秘配方"本质上是一种营销叙事，核心精油配方并不难复制，真正的秘密在于工业化生产的一致性。

2. **精油来源与安全性讨论**：化学背景的用户讨论了精油的来源（食品级 vs 香薰级）、肉桂油的安全使用量等问题。有人提醒未稀释精油具有腐蚀性，文章的安全警告是必要的。

3. **"为什么要自己做"的价值观讨论**：有人问为什么要费这么多功夫做可乐，作者（和支持者）的回答是：理解食物的来源、控制成分（无咖啡因、无特定甜味剂）、纯粹的创作乐趣——这与写代码的动机如出一辙。一位用户写道："这和从头自己写框架是一样的心态。"

4. **气泡水设备讨论**：评论区出现了大量关于 SodaStream、Aarke 以及自建 CO₂ 充气系统的实用建议，有人详细介绍了如何用焊接级 CO₂ 气瓶替代昂贵的 SodaStream 气罐来降低成本。
