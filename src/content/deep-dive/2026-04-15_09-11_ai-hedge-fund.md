# 11 · virattt/ai-hedge-fund

> ⭐ 3,402 stars / week | 54,143 total | 2026-04-15

## a. 定位与痛点剖析

**ai-hedge-fund** 是一个教育性质的 AI 驱动对冲基金模拟系统，核心是多 Agent 协作框架模拟不同投资大师的决策风格。

痛点：量化投资的"多因子分析"本身非常适合 Multi-Agent 架构，但传统量化框架（backtrader、zipline）对新手不友好，无法体现不同投资哲学的差异。这个项目通过**拟人化投资 Agent**让投资逻辑可解释、可学习。

**重要免责**：这是一个纯教育/研究项目，不进行真实交易，不提供投资建议。

## b. 核心架构与技术细节

**技术栈**：Python + Poetry，支持 OpenAI/Groq/Anthropic/DeepSeek/Ollama（本地模型）

**Agent 团队（20个）**：
- **投资哲学 Agents（13个）**：
  - 价值投资：Ben Graham（安全边际）、Charlie Munger（优质公司）、Warren Buffett（护城河）
  - 成长投资：Cathie Wood（颠覆性创新）、Phil Fisher（深度调研）
  - 对冲/宏观：Stanley Druckenmiller（宏观非对称）、Nassim Taleb（黑天鹅/尾部风险）、Michael Burry（深度价值逆向）
  - 印度市场：Rakesh Jhunjhunwala（"印度股神"）
  - 估值专家：Aswath Damodaran（"估值之王"）
  - 激进主义：Bill Ackman（激进投资者）
  - Mohnish Pabrai（Dhandho 框架）、Peter Lynch（十倍股）

- **分析 Agents（4个）**：Valuation（内在价值）、Sentiment（市场情绪）、Fundamentals（基本面）、Technicals（技术面）

- **决策 Agents（3个）**：Risk Manager（风险指标+仓位限制）、Portfolio Manager（最终决策+订单）

**执行方式**：
```bash
poetry run python src/main.py --ticker AAPL,MSFT,NVDA
poetry run python src/backtester.py --ticker AAPL,MSFT,NVDA  # 回测
```

**Web 应用**：提供可视化界面，比 CLI 更友好，支持实时流式输出

## d. 开发者反馈与局限性

**正面**：
- 54k total stars，长期稳定增长，成为 Multi-Agent 入门教程的标杆
- 支持 Ollama 本地模型，隐私和成本敏感用户友好
- 代码结构清晰，适合学习 LangGraph/多 Agent 编排模式

**局限性**：
- 仅教育用途（README 三次强调"不用于真实交易"），商业应用价值有限
- Agent 的"投资哲学"模拟本质上是 prompt engineering，未必准确反映真实大师的决策逻辑
- 数据来源（FINANCIAL_DATASETS_API_KEY）有成本，免费额度有限
- 没有风险管理的深度实现，更多是展示 Agent 协作框架
- Portfolio Manager 的"最终决策"过程缺乏透明度
