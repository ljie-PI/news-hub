## 背景

2026 年的网页设计正在被系统性地"压缩"——几何结构、排版节奏、文案语气与调色板共同构成了一个站点的视觉指纹，被业内统称为 design DNA。随着 LLM 与多模态模型逐渐具备"读图—解构—复现"的端到端能力，把任意 URL 提炼成一组结构化、可复用的设计 token，已经从设计师的私房手艺演变为 UI Agent 工作流的标配前置工序。Taste Lab 以 267 票登上 PH 当日榜，正是这股趋势的最新落地标志。

## 产品机制

输入任意网址后，Taste Lab 会爬取首屏与若干子页，输出五类结构化资产：(1) 主/辅调色板与对比度比例；(2) 字体家族、字重与排版尺度阶；(3) spacing scale 与栅格密度；(4) 品牌 voice（slogan 句式、CTA 用词倾向、情感色彩）；(5) 可重用 component 截图与近似命名。结果可一键导出为 Tailwind config、Figma Variables 或 JSON token，直接喂给 Cursor、v0、Claude Code 复刻或二创新站，缩短了从"看到一个喜欢的站"到"动手实现"的距离。

## 社区反应

## 行业影响

Figma Make、Galileo、Uizard 走的是"prompt→生成新设计"的合成路线，Taste Lab 则反向切入"既有站→结构化 DNA"，更贴近 design system extraction 这一 niche。对 UI Agent 而言，Taste Lab 输出的 token 化风格上下文，正是 Cursor、v0 等代码 agent 长期缺失的拼图，未来极可能以 MCP server 形态嵌入主流 IDE，成为多模态 design-to-code 链路的标准前置组件。
