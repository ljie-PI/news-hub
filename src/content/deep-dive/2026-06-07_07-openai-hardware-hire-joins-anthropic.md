## 1. 事件背景

6 月 7 日,r/ClaudeCode 上一张 OpenAI 员工跳槽 Anthropic 的截图火了。原帖配文写道:"Didnt know they called stock packages mountains these days but pretty interesting 😅"——指被挖角员工拿到的"山一般"的股票包。这是过去半年 Anthropic 从 OpenAI 持续挖角的最新一例:5 月 Andrej Karpathy 已正式加入 Anthropic 预训练团队,Ross Nordeen 等多位 OpenAI/xAI 前成员也接连转投。值得关注的是,这次跳槽的方向是 **Hardware**,直接对应 Anthropic 与 Google/Broadcom 合作部署 3.5 GW TPU 集群的硬件路线。

## 2. 核心观点 / 产品机制

帖子核心信息可以拆为三层:① **个人决策**——OpenAI 硬件序列的工程师离开 Sam Altman 团队;② **公司布局**——Anthropic 正从 GPU 单一路线扩张到 TPU + 自研加速器路线,与 Karpathy 主导的"用 Claude 加速 pretraining"形成软硬协同;③ **薪酬信号**——所谓"mountain-sized stock package"反映出 Anthropic 用最新一轮 30B+ 美元融资后的估值,在头部硬件人才上开出与 Meta、xAI 同档甚至更高的 offer。Vibe Coding 圈关心的不只是八卦,更是 Claude Code 背后能否持续拿到便宜算力。

## 3. 社区热议与争议点

- **正面 A(用脚投票)**:有评论写道 "Anthropic 招得到 OpenAI 硬件人,说明他们对长期 roadmap 比 OpenAI 更有信心",把人员流向解读为研究文化与安全文化的胜利。
- **正面 B(Claude Code 用户角度)**:Vibe Coding 玩家盼着 Anthropic 算力扩张能压低 Claude Code 的限速,"More TPU = less rate limit on Sonnet"成了高赞调侃。

## 4. 行业影响与未来展望
