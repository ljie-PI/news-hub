## 事件背景

据《华尔街日报》6月13日报道，亚马逊 CEO Andy Jassy 私下向财政部长 Scott Bessent 等白宫官员通报：Amazon 内部研究人员通过一系列 prompt 成功越狱 Anthropic 的 Fable 5 模型，提取出可用于网络攻击的禁区信息。这场对话直接触发了特朗普政府的出口管制指令——6月12日 Anthropic 被迫对**所有客户**（包括自家外籍员工）停用最新的 Fable 5 与 Mythos 5 两款旗舰模型。Fable 5 是 Anthropic 6月9日才广泛开放的首款 Mythos 级前沿模型，问世仅三天即遭封禁。

## 核心争议点

最讽刺的是，举报者正是 Anthropic 最大金主。截至2026年2月，Amazon 对 Anthropic 持有 458亿可转债 + 148亿优先股，敞口逾 600亿美元，占股本5%以上。但 Amazon 自家 Bedrock 平台与 Anthropic 直营 API 存在直接竞争，且 Bezos 新创公司也意图用 AI 垄断重工程领域。"自己投资、自己越狱、自己举报"的悖论，让外界普遍质疑：这究竟是真心忧虑 AI 安全，还是借政府之手压制被投企业、迫其"屈膝"？

## 行业影响

事件给 LLM 竞争格局投下深水炸弹：其一，**头部美国厂商首次被同行+政府联手"点名"下架**，开创危险先例，企业客户对单一闭源 API 的依赖风险骤升；其二，Microsoft 早已抱怨 Fable 缺乏 ZDR 不能用，如今禁令更让 Azure/Bedrock 上的 Anthropic 流量陷入合规真空；其三，海外用户和外籍工程师被一刀切，反向推高了对 **GLM-5.1、Kimi、Llama、DeepSeek** 等开源/非美权重替代品的刚需——评论区已有人公开用 GLM-5.1 改写 refusal 训练数据。可以预见，未来 6-12 个月将出现一波"去 Anthropic 化"的多模型路由与开源自托管浪潮。
