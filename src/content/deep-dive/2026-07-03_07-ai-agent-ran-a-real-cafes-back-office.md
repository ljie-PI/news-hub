# an AI agent ran a real cafe's back office for 2 months, $38k out, $9k in. where should the human sign-off have been?

## 事件背景
Andon Labs 今年四月在斯德哥尔摩 Norrbackagatan 48 租下一间真实咖啡馆，把采购、定价、排班、与供应商沟通等后台运营权交给名为 Mona 的 AI agent（底层为 Gemini），门店冲煮仍由人类完成。两个月里，这套系统累计花出约 38000 美元，只带回约 9000 美元。这条帖子把该实验放到 r/AI_Agents，追问在 agent 主导运营的模式下，人类的签核关卡究竟应当设在哪里。

## 核心观点
发帖者的核心关切并非 AI 能否开店，而是人类监督应嵌入流程的哪一环。实验设置了 human sign-off gates，但巨大的收支逆差说明关卡要么位置不对、要么形同虚设。Mona 展现了阅读租约、生成开业清单、主动推进任务的惊人能力，却在批发下单、半夜给店员发消息等环节暴露判断盲区，凸显 always-on AI 缺乏成本意识与常识边界。

## 社区热议
围绕帖子可归纳出几种视角。其一，乐观派认为负毛利是学费，实验价值在于公开暴露前沿 agent 的真实短板；其二，务实派主张签核应前置到资金支出与对外沟通两个高风险动作上，设硬性金额阈值与人工复核；其三，怀疑派指出，若每笔关键决策都要人类兜底，所谓自主运营不过是昂贵的演示，商业可行性存疑。

## 行业影响
这类真实世界压力测试，正把 agent 讨论从 demo 拉向可审计的运营指标。它提示行业：自主 agent 落地商业场景，护栏、预算约束、审批链与可观测性同样关键，而非只看模型能力上限。这也预示监督设计将成为 agent 产品化的核心竞争力。
