# 48小时用Claude Code做出Balatro风格飞镖游戏：Vibe Coding游戏开发的新范式

这一案例出现的时间节点恰好处于"Vibe Coding"（氛围编程）概念爆发期。2026年初以来，越来越多的非专业程序员和独立开发者通过Claude Code、Cursor等AI编程工具快速产出游戏项目，r/vibecoding已成为此类分享的核心社区。

## 2. 核心观点与产品机制

该项目的核心卖点在于**速度与完成度的反差**：48小时的开发周期对于一个包含Roguelike机制、卡牌增益系统和飞镖投掷玩法的游戏而言极为紧凑。开发者很可能采用了以下工作流：

- **Claude Code作为主力编程工具**：通过自然语言描述游戏逻辑，让AI生成核心代码框架
- **借鉴Balatro的成熟设计模式**：回合制得分目标、Joker式增益卡牌、分数乘数机制，这些经过市场验证的设计降低了游戏设计的试错成本
- **飞镖主题的差异化创新**：将扑克牌替换为飞镖投掷，靶盘区域对应不同基础分值，保留了策略深度的同时引入了操作技巧元素

这种"成熟机制 + 新主题 + AI加速"的组合正在成为Vibe Coding游戏开发的标准公式。
